B. O que é arquitetura cliente-servidor utilizada na web ? Explique o carregamento de uma página passo a passo

R: A arquitetura cliente-servidor na web refere-se à distribuição de funções entre um cliente e um servidor. O cliente solicita recursos, e o servidor fornece esses recursos. O carregamento de uma página web se baseia nos seguintes passos: O navegador (cliente) envia uma solicitação HTTP para o servidor, depois o servidor processa a solicitação e envia de volta os arquivos necessários, como HTML, CSS e JavaScript e então o navegador interpreta o HTML para construir a estrutura da página assim solicitando recursos adicionais, como imagens, scripts e folhas de estilo e depois renderizando a página, exibindo-a para o usuário.
